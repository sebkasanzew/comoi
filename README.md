Comoi is a grocery marketplace connecting consumers with local mini-markets in Southeast Asia, starting in Vietnam.

## Overview
- Location-based market discovery
- Price comparison across nearby vendors
- Pickup or delivery ordering
- Mobile-first experience with offline tolerance
- Vendor-friendly catalog and order tools

## Tech stack (high level)
- Web: Next.js (App Router)
- Mobile: React Native + Expo
- Backend: Convex
- Business logic: Effect
- UI: React
- Monorepo: Turborepo
- Package manager: Bun
- Linting: Biome + ESLint
- Testing: Vitest, Playwright, Detox
